enum SBSDKShutterMode

This enum describes the possible modes of the cameras shutter. Automatic shutter means, a photo is taken automatically if a document was detected in the video stream. The smart mode is the default mode. Whenever the automatic shutter is toggled the delegate of SBSDKScannerViewController is informed. You can also the query the autoShutterEnabled property. The default shutter button reflects this mode.

Enumerator
SBSDKShutterModeSmart 

Toggles the automatic shutter in a smart way. If there, for 3 seconds, is no significant device motion and no document was detected the automatic snapping is turned off. Significant device motion turns it on again.

SBSDKShutterModeAlwaysAuto 

The camera will always take a photo automatically when a document was detected.

SBSDKShutterModeAlwaysManual 

The camera will never take a photo automatically.
